# Rahul Kumar Portfolio

This repository contains the source code for my personal portfolio website built with HTML, CSS, and JavaScript.

## Overview

The site showcases:
- my professional background as a Senior Data Engineer
- key technical skills and tools
- featured projects and impact metrics
- certifications and additional professional highlights
- a downloadable resume

## Files

- index.html - main structure of the portfolio website
- styles.css - styling and layout
- script.js - small interactive behavior
- rahulSoftwareEngineerCV.pdf - downloadable resume

## Preview locally

Open index.html in your browser, or serve the folder with a simple local server if you prefer.
